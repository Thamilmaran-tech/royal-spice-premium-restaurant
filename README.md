## 🌟 Overview

**Royal Spice Premium Restaurant** is a modern multi-page restaurant website designed to provide an immersive digital dining experience.

The project combines elegant design, responsive layouts, and smooth navigation to showcase a premium restaurant brand. Built using **HTML5**, **CSS3**, **Bootstrap 5**, and **JavaScript**, it demonstrates real-world frontend development practices and modern UI principles. Bootstrap is widely used for responsive, mobile-first web development. :contentReference[oaicite:0]{index=0}
